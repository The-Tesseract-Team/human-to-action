# Support

This is a volunteer, non-commercial hobby project. Support is best effort and has no service-level commitment.

Before opening an issue:

1. Confirm you installed the intended marketplace and plugin version.
2. Start a new Codex task after installation or update.
3. Record your operating system, Codex surface, plugin version, expected behavior, and actual behavior.
4. Remove prompt content, credentials, personal data, and private project details.

Use the repository's bug or feature issue form for public support. Use the private process in [SECURITY.md](SECURITY.md) for security or privacy concerns.
