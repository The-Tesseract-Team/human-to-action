# Code of Conduct

## Our commitment

We are committed to a welcoming, respectful, and harassment-free project for contributors of every background and experience level.

## Expected behavior

- Be respectful, specific, and constructive.
- Critique ideas and code rather than people.
- Respect privacy, consent, and security boundaries.
- Accept correction and make a good-faith effort to repair harm.
- Keep project discussions relevant and avoid spam or intimidation.

Harassment, discrimination, threats, deliberate disclosure of private information, sexualized conduct, and sustained disruption are not acceptable.

## Scope and enforcement

This code applies in repository issues, pull requests, discussions, reviews, and other spaces where someone represents this project. Maintainers may edit or remove contributions, restrict participation, or report serious abuse to the hosting provider.

Report conduct concerns privately to an organization owner when possible. If no appropriate private project channel is available, use [GitHub's abuse reporting process](https://support.github.com/contact/report-abuse). Do not expose personal or sensitive details in a public issue.

This volunteer project handles reports on a best-effort basis and will prioritize safety, privacy, context, and proportional action.
